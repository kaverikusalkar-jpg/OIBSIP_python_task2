# OIBSIP_python_task2
Password generator using python
Objective:
To generate strong, customizable random passwords using Python  and HTML-CSS-JavaScript (web) by allowing users to choose length and character types.

Tools & Outcome:
Tools used are Python (random, string) and Web technologies (HTML, CSS, JavaScript). The outcome is a secure password generator that works both in the terminal and in a user-friendly browser interface.
